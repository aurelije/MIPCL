§ MIP_API

#define MIP_API

WINDOWS specifics.

All files within this DLL are compiled with the MIP_EXPORTS symbol defined on the command line. This symbol should not be defined on any project that uses this DLL. This way any other project whose source files include this file see MIP_API functions as being imported from a DLL, whereas this DLL sees symbols defined with this macro as being exported.
