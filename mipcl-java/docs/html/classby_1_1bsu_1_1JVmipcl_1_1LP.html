◆ getSolution()

native void by.bsu.JVmipcl.LP.getSolution ( )

The function gets from the CLP object

  • variable handles (stores them in varHd);
  • constraint handles (stores them in ctrHd);
  • values of variables (stores them in X);
  • reduced costs (stores them in rdCost);
  • shadow prices (stores them in shPrice).
    Remarks
    The order in which the variables are listed in varHd is not necessary the same they were added to the matrix by addVar() or addColumn(). Similarly, the order in which the constraints are listed in ctrHd is not necessary the same they were added to the matrix by addCtr() or addRow().
    See also
    MIP.getSolution().
