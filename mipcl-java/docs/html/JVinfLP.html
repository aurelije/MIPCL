Why an LP has no solution?

Certificates of Infeasibility and Boundlessness

After having been solved an LP, we usually call CLP::isSolution() to check whether a solution was found, and, if it turns out that the return value is false, then our LP has no solution. Next, we will want to know why this LP has no solution. We say that an LP has no solution if either

  • either 1) its system of inequalities is inconsistent (has no solution)
  • or 2) its objective is unbounded over the set of feasible solutions.

If LP.isSolution() returns false, we know that our LP has no solution and then we can call LP.isLpInfeasible() and/or LP.isLpUnbounded() to know whether this LP has no feasible solution (isLpInfeasible() returns true) or its objective is unbounded (isLpUnbounded() returns true).

To get certificates of inconsistency and boundlessness, we call the following functions: LP.whyLpInfeasible() and LP.whyLpUnbounded().

As an example of an LP that has no feasible solution, we consider the problem of maximizing subject to the constraints (1). The following procedure solves this simple LP.

import by.bsu.JVmipcl.*;
public class InfLP
{
static {
System.loadLibrary("JVmipcl");
}
public static void main(String[] args) {
try {
LP lp = new LP("infLP");
lp.openMatrix(2,3,5);
lp.addVar(0,0,4.0,0.0,LP.VAR_INF);
lp.addVar(1,0,1.0,0.0,LP.VAR_INF);
lp.addVar(2,0,2.0,0.0,LP.VAR_INF);
lp.addCtr(0,0,-LP.INF,5.0);
lp.addCtr(1,0,3.0,LP.INF);
lp.addEntry(2.0,0,0);
lp.addEntry(5.0,0,1);
lp.addEntry(1.0,0,2);
lp.addEntry(1.0,1,0);
lp.addEntry(2.0,1,1);
lp.preprocOff();
lp.closeMatrix();
lp.optimize();
lp.printSolution("infLP.sol");
}
catch(Throwable e) {
System.out.println(e);
}
}
}

If we compile and then run (see JVmipcl Primer to know how to do this) the above program, then we get the output file infLP.sol whose content is as follows.

Constraints are conradictory!
Infeasibility Certificate:
ctr_0: 1 * (2*x(0) + 5*x(1) + 1*x(2) <= 5) +
ctr_1: -2 * (3 <= 1*x(0) + 2*x(1)) +
-1 * (0 <= x(1)) +
-1 * (0 <= x(2)) =
0 <= -1

In fact, the solver has found the same proof of inconsistency for (1) that we had found earlier.

We see that in case when the constraint system of an LP is inconsistent, then the function LP.printSolution() outputs a certificate of inconsistency. In fact, LP.printSolution() calls another function, LP.showWhyLpInfeasible(), that writes to the output file a certificate of inconsistency.

Similarly, when the objective of an LP is unbounded, LP.printSolution() calls the function LP.showWhyLpUnbounded() to write a certificate of boundlessness to the output file.
