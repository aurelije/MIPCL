Detailed Description

A constraint is a linear sum with lower and upper bounds. A linear sum (object of type CLinSum) is a sum of linear terms, while a linear term is either a real constant or the product of a real constant and a variable. In MIPSHELL, linear terms, linear sums, and constraints are implemented as objects of types CTerm, CLinSum, and CCtr, respectively.
