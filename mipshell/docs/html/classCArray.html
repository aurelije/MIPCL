Detailed Description

template<typename Tell>
class CArray< Tell >

In MIPshell dense arrays are called vectors, while sparse arrays (with many zero/empty entries which are not stored) are simply called arrays and are represented as objects of class CArray.

See also
MIPshell Arrays.
