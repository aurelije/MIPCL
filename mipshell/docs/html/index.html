A language for writing Mixed Integer Programs
MIPschell library

Anyone who has ever attempted to apply mathematical programming in practice knows that it is usually not a simple and straightforward exercise. MIPshell is an environment that facilitates modeling and solving linear and mixed-integer programming problems with MIPCL. At this moment, MIPshell is not just a stable programming product. Nevertheless, it has been used for modeling a great deal of test as well as practical problems.

MIPshell comprises

  • collection of C++ classes that represent variables, constraints, sets, vectors, arrays, and etc.;
  • library of functions to simplify posing optimization problems;
  • preprocessor that translates new modeling operators into fragments of C++ code.

To help you get started quickly, MIPshell comes with

  • MIPschell Primer that gives a brief introduction to MIPshell;
  • a set of documented examples
    • Product Mix
    • Fixed Charge Network Flows
    • Balancing Assembly Lines
    • Service Facility Location
    • Process Layout
    • Multiproduct Lot-Sizing
    • Electricity Generation Planning
    • Short-Term Scheduling in Chemical Industry
    • Portfolio Optimization (demonstrates generation of cuts)
    • Yield management Revenue Management In Airline Industry (demonstraits scenario approach for solving stochastic programming problems)
  • and non-documented examples that can be found in the $MIPCLDIR/examples/mipshell folder, where the environment variable MIPCLDIR is set to point to the MIPCL installation directory.
