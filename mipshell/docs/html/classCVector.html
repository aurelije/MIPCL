Detailed Description

template<typename Tell>
class CVector< Tell >

In MIPshell a vector is multidimensional array which indices are integers starting from 0.

The constructor creates an empty array which dimension is the number of its arguments with positive value. For example, the declaration

CVector<double> a(3,2);

creates a real 2-dimension vector of size 2x3.

Parameters
[in]size0,size1,size2,size3sizes of each of four possible dimensions; zero sizes may be skiped.
