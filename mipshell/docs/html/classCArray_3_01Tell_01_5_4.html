Detailed Description

template<typename Tell>
class CArray< Tell *>

Specialization of CArray<Tell> for pointers.

This specialization is useful in those cases when array elements are big in size and they were previously stored in some other data structure. It would be inefficient to copy those elements into the array. Instead, it is better to store in the array only pointers to the elements.
